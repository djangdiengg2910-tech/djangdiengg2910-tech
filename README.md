## 🎮 Featured Project: Knowledge Guesser — AI-Powered Quiz Game

A minimalist quiz game where players discover hidden keywords through AI-generated hints. The fewer hints used, the higher the score.

### Key Features

* 🧠 Dynamic hint generation using Gemini API
* 🎯 Multiple categories: science, history, technology, culture, and more
* 📈 Adaptive scoring system based on hints used
* ⚡ Lightweight implementation using pure HTML, CSS, and JavaScript

### Tech Stack

* Architecture: Client-side state management
